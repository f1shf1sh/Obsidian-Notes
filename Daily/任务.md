# 主要方向
以 RoCEv2 作为核心研究对象，重点关注其在主流商用实现中的关键技术特征，系统分析RoCEv2 在RDMA场景下，尤其是面向GPU直连场景中的技术细节
# 总体研究目标
1. 系统梳理当前支持 RDMA 技术的主流硬件平台与可二次开发的开源产品生态，明确不同硬件方案在 RoCEv2 场景下的适配能力与可扩展性
2. 调研并分析现有RoCEv2相关的开源软件栈与仿真框架
3. 搜集国内外从事RDMA相关研究的团队信息，以及对应团队的代表性论文，明确各团队的研究方向与核心成果
# 具体调研任务
 1. RDMA 硬件与开源产品调研：
	 系统梳理国内支持RDMA技术的硬件设备，重点分析其对 RoCEv2的支持情况、可二次开发能力、部署成本及典型应用场景，同时调研相关开源RDMA产品与软件项目。
2. RoCEv2 相关开源框架与仿真平台调研：
	在主流开源平台上检索RoCEv2相关的软件栈与仿真框架
3. 国内外研究团队与论文调研：
    系统整理国内外从事RDMA/RoCE相关研究的高校、科研机构与工业界团队，梳理其代表性论文与核心研究成果
4. iRoCEv2/IB/iWARP三大协议对比表：
    主要关注内容：核心原理、传输速率、延迟性能、兼容性、部署成本、细分应用场景、优势、劣势、可扩展性等
