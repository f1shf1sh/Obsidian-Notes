## 本轮补充内容

- 第一章新增了有效数字判据推导、误差传播公式推导，以及“避免相近数相减”“面积误差传播”两个经典例题。
- 第二章新增了二分法误差界、不动点局部收敛、Newton 二阶收敛、重根退化等推导，并增加了二分法、Newton 法、重根修正三个经典例题。
- 第三章直接法新增了从消元过程到 LU 分解的推导、$PA=LU$ 的解释，以及 LU 分解求解和选主元必要性的经典例题。
- 第三章迭代法新增了 Jacobi / Gauss-Seidel / SOR / 最速下降 / CG 的关键推导，并加入了 Jacobi 与 Gauss-Seidel 对比、CG 两步求解的经典例题。
- 第四章新增了特征值问题导入、Gerschgorin 圆盘定理、幂法、幂法加速、反幂法、移位反幂法的详细总结，并补了经典例题。
- 第五章新增了 Lagrange 插值、插值误差、Runge 现象、分段低次插值、Newton 均差表、Hermite 插值和三次样条插值的冲刺复习总结。
- 第六章新增了最小二乘拟合、正规方程组、可线性化非线性拟合、正交多项式、Legendre / Chebyshev / Hermite 多项式和最佳平方逼近的冲刺复习总结。
- 第七章新增了数值积分概述、代数精度、Newton-Cotes 公式、复合求积公式、Romberg 求积、Gauss 求积和数值微分的冲刺复习总结。
- 第八章新增了 Euler 方法、后退 Euler 方法、梯形方法、改进 Euler 方法、Runge-Kutta 方法、单步法收敛稳定性、线性多步法和高阶方程化一阶方程组的冲刺复习总结。
- 总复习文件新增了 1-8 章重要公式速查、每章代表性经典大题和简版参考答案，适合考前最后一轮快速过题。
- 强化训练文件按研究生期末大题难度补充了 1-8 章高频大题知识点，每个重要考点至少配一道综合题并附参考答案要点。

## 使用建议

1. 先看每份笔记开头的图示导读，建立方法直觉。
2. 再按正文梳理定义、定理、算法格式与误差公式。
3. 最后重点背诵各章末尾整理出的“必须掌握的公式”。
