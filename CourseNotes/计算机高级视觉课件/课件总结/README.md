# 计算机高级视觉课件总结

本目录将 `./计算机高级视觉课件` 中现有 PDF 课件整理成了按讲次划分的 Markdown 复习笔记，统一放在这里，方便连续复习。

## 目录说明

- `01_课程导论.md`：对应 `master-L1_Introduction-Bachelor.pdf`
- `02_光与颜色.md`：对应 `L2_LightAndColor.pdf`
- `03_图像滤波.md`：对应 `L3_ImageFiltering.pdf`
- `04_频域分析与傅里叶思维.md`：对应 `L4-5_ThinkingInFrequency.pdf`
- `05_边缘检测.md`：对应 `L7_EdgeDetection.pdf`
- `images/`：笔记中引用的关键课件截图

## 使用建议

1. 先看每个 Markdown 的“本讲主线”，建立整体框架。
2. 再记公式，尤其是卷积、傅里叶变换、梯度和 Canny 流程。
3. 最后结合课件截图回忆讲课时的例子和图示。

## 复习顺序建议

1. 课程导论
2. 光与颜色
3. 图像滤波
4. 频域分析与傅里叶思维
5. 边缘检测

之所以按这个顺序，是因为：

- 图像表示、光照和颜色是图像形成基础；
- 滤波是后面频域分析和边缘检测的前置；
- 边缘检测本质上依赖滤波、导数和频域直觉。
