# 数值分析课件图文复习总结

本目录把 `数值分析课件` 中的 PDF 重新整理为独立的 Markdown 复习资料，并统一放到一个目录中，便于按章节复习。

## 文件索引

- `01_第一章_引论.md`
- `02_第二章_非线性方程求根方法.md`
- `03_第三章_线性方程组的数值解法（一）_直接法.md`
- `04_第三章_线性方程组的数值解法（二）_迭代法.md`

## 配图说明

- 课件截图统一放在 `images/` 目录。
- 每份笔记开头都有“课件图示导读”，方便先看图再看公式。
- 第三章两份笔记中额外补充了矩阵理论相关说明，包括顺序主子式、谱半径、对称正定、条件数、$A$-共轭等概念。

## 本轮补充内容

- 第一章新增了有效数字判据推导、误差传播公式推导，以及“避免相近数相减”“面积误差传播”两个经典例题。
- 第二章新增了二分法误差界、不动点局部收敛、Newton 二阶收敛、重根退化等推导，并增加了二分法、Newton 法、重根修正三个经典例题。
- 第三章直接法新增了从消元过程到 LU 分解的推导、$PA=LU$ 的解释，以及 LU 分解求解和选主元必要性的经典例题。
- 第三章迭代法新增了 Jacobi / Gauss-Seidel / SOR / 最速下降 / CG 的关键推导，并加入了 Jacobi 与 Gauss-Seidel 对比、CG 两步求解的经典例题。

## 使用建议

1. 先看每份笔记开头的图示导读，建立方法直觉。
2. 再按正文梳理定义、定理、算法格式与误差公式。
3. 最后重点背诵各章末尾整理出的“必须掌握的公式”。
